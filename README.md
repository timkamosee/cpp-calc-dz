# C++ Calculator

Консольный калькулятор на C++ с поддержкой коротких и длинных аргументов командной строки через `getopt_long`.

## Сборка

```bash
cmake -B build
cmake --build build
```

После сборки бинарник появится в директории `build/`.

Запуск:

```bash
./calculator [OPTIONS]
```

---

# Аргументы командной строки

| Короткий | Длинный  | Описание             |
| -------- | -------- | -------------------- |
| `-a`     | `--arga` | первый операнд       |
| `-b`     | `--argb` | второй операнд       |
| `-h`     | `--help` | показать справку     |
| —        | `--add`  | сложение             |
| —        | `--sub`  | вычитание            |
| —        | `--mul`  | умножение            |
| —        | `--div`  | деление              |
| —        | `--pow`  | возведение в степень |
| —        | `--fact` | факториал            |

---

# Примеры использования

### Сложение

```bash
./calculator -a 5 -b 3 --add
```

или

```bash
./calculator --arga 5 --argb 3 --add
```

Результат:

```
8
```

---

### Вычитание

```bash
./calculator -a 10 -b 4 --sub
```

```
6
```

---

### Умножение

```bash
./calculator -a 6 -b 7 --mul
```

```
42
```

---

### Деление

```bash
./calculator -a 20 -b 5 --div
```

```
4
```

---

### Возведение в степень

```bash
./calculator -a 2 -b 10 --pow
```

```
1024
```

---

### Факториал

Для факториала используется только один аргумент.

```bash
./calculator -a 5 --fact
```

```
120
```

---

# Ошибки ввода

Программа выводит ошибку если:

* отсутствуют необходимые аргументы
* передано несколько операций одновременно
* переданы лишние аргументы
* указана неизвестная опция
