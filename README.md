# Calculator (C++ CLI)

Консольный калькулятор на **C++17**, поддерживающий базовые арифметические операции, возведение в степень и факториал.
Проект использует **CMake**, **clang-tidy**, **clang-format** и подключает математическую библиотеку через **FetchContent**.

## Возможности

Поддерживаемые операции:

| Операция | Описание                                       |
| -------- | ---------------------------------------------- |
| `+`      | Сложение                                       |
| `-`      | Вычитание                                      |
| `*`      | Умножение                                      |
| `/`      | Деление                                        |
| `^`      | Возведение в степень (итеративно)              |
| `!`      | Факториал (рекурсивно, использует только `-a`) |

Программа проверяет:

* переполнение типа
* переполнение после вычисления
* деление на ноль
* некорректный ввод
* факториал отрицательного числа

# Сборка проекта

## Требования

* CMake ≥ 3.20
* C++17 компилятор (g++, clang++)
* clang-tidy
* clang-format
* git

## Сборка

```bash
git clone https://github.com/timkamosee/calc.git
cd calc

cmake -B build
cmake --build build
```

Исполняемый файл появится:

```
build/calculator
```

---

# Использование

Формат запуска:

```
calculator -a <number> -b <number> -o <operation>
```

или

```
calculator -h
```

### Параметры

| Флаг | Описание     |
| ---- | ------------ |
| `-a` | первое число |
| `-b` | второе число |
| `-o` | операция     |
| `-h` | справка      |

---

# Примеры

Сложение

```
./calculator -a 5 -b 3 -o +
```

Результат:

```
8
```

Возведение в степень

```
./calculator -a 2 -b 10 -o ^
```

Результат:

```
1024
```

Факториал

```
./calculator -a 5 -o !
```

Результат:

```
120
```

# Обработка ошибок

Программа может выводить следующие ошибки:

| Код | Описание                       |
| --- | ------------------------------ |
| 1   | Некорректный ввод              |
| 2   | Переполнение при вводе         |
| 3   | Деление на ноль                |
| 4   | Переполнение после вычисления  |
| 5   | Вызов справки                  |
| 6   | Факториал отрицательного числа |
